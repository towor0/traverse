Requirements:
Python
Pygame

Instructions:
Run traverse.py to start the game

Description:
Play as a cave explorer who's too far gone. 
The only path is to go down. 
Blinded, monsters are also overwhelmed, use sound cues to help you navigate the rudimentary caves.
Be careful, monsters are also listening, and don't make any sounds while nearby.
In extreme cases, blood could be drawn to mark areas or to distract the monsters.

Controls:
Mouse Movement - Changes Player's Angle
Mouse Click - Bleed
W - Forward Movement
S - Backward Movement
A - Left Movement
D - Right Movement
Left Shift - Sprint
